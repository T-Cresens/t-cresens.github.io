Side project — 2026

SPIKE

v1.0.001

Seizure Pattern Intelligence & Knowledge Engine — a Windows desktop tool for EEG pattern analysis and epilepsy decision support.

Download — Windows GitHub

What it does

Reads clinical EEG files (.edf), applies pattern detection models trained on public datasets (TUH, CHB-MIT, Siena, TUEP), and produces an epilepsy probability estimate with a conformal prediction interval. Includes a decision-support tab for AED selection based on Belgian and NICE guidelines. Built in Python with MNE-Python, LightGBM, and MAPIE.
